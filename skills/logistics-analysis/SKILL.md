# 整车物流调度分析 Skill

用于分析整车物流公司的车队运营数据，生成供需平衡表和调度优化建议。

## 适用场景

✅ **使用此Skill的场景：**

- 分析车队运输数据，生成运营报告
- 计算各车队供需平衡，发现资源错配
- 导入Excel/CSV格式的路由池明细表
- 生成可视化看板（地图、饼图、条形图）
- 获取调度优化建议

## 数据要求

### 输入文件格式

支持以下格式：
- Excel (.xlsx, .xls)
- CSV (.csv)

### 必需列名

| 列名 | 说明 |
|------|------|
| 大区 | 东北/华北/华东/华中/华南/西南/西北 |
| 省公司 | 业务中心名称 |
| 品牌 | 汽车品牌 |
| 承运商名称 | 车队名称 |
| 始发省 | 起始省份 |
| 目的省 | 目的省份 |

### 示例数据

```
大区,省公司,品牌,承运商名称,始发省,目的省
华南公司,广州业务中心,小鹏汽车,广州车队,广东省,广东省
华东公司,常熟业务中心,理想汽车,常熟车队,江苏省,上海
华北公司,北京业务中心,北京现代,顺义车队,北京,天津
```

## 功能说明

### 1. 数据导入与分析

```python
# 分析Excel数据
python3 logistics_analysis.py --file 路由池明细表.xlsx --analyze

# 分析CSV数据  
python3 logistics_analysis.py --file data.csv --analyze
```

### 2. 生成报告

```python
# 生成完整分析报告
python3 logistics_analysis.py --file data.xlsx --report

# 生成供需平衡表
python3 logistics_analysis.py --file data.xlsx --balance
```

### 3. 可视化看板

```python
# 生成HTML可视化看板
python3 logistics_analysis.py --file data.xlsx --visualize

# 打开看板
open logistics_dashboard.html
```

## 输出指标

### 基础统计

- 总运单数
- 大区分布
- 品牌分布
- 车队分布

### 高级分析

- **供需平衡表**：各车队当前运力 vs 区域需求
- **线路统计**：各车队运营线路数、跨省比例
- **品牌集中度**：各业务中心品牌依赖度
- **优化建议**：基于数据的调度优化方向

## 调度优化建议模板

基于分析结果，会自动生成以下建议：

### 🔴 紧急 - 资源调配

1. **跨区支援建议**
   - [车队A] → [目的地B]，运力缺口X单
   
2. **回流线路优化**
   - 设计 [A→B] + [B→A] 对流线路

### 🟡 中期 - 流程优化

3. **品牌分散建议**
   - 减少对单一品牌的依赖

4. **线路整合建议**
   - 合并相似线路，提高单车效率

### 🟢 长期 - 战略规划

5. **车队组网建议**
   - 按大区建立调度中心
