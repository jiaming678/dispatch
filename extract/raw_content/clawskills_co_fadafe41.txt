## Frequently Asked Questions
What are OpenClaw Skills?
OpenClaw Skills are community-built extensions for AI coding agents. They add new capabilities like connecting to APIs, automating workflows, following coding standards, and more. Skills are stored as markdown files and can be installed globally or per-project.
How do I install a skill?
Three ways: (1) Run `npx clawhub@latest install ` in your terminal. (2) Paste the skill's GitHub link into your AI assistant's chat and ask it to install. (3) Manually copy the skill folder to `~/.openclaw/skills/`.
Are OpenClaw Skills free?
Yes. All skills listed on ClawSkills and in the awesome-openclaw-skills repository on GitHub are free and open source. Anyone can create and share skills with the community.
What AI agents work with OpenClaw Skills?
Skills work with OpenClaw, Claude Code, Codex CLI, OpenCode, and other AI coding agents that support the OpenClaw skill format. Skills stored in workspace or global directories are automatically loaded.
Where can I find more skills on GitHub?
The main community repository is [awesome-openclaw-skills](https://github.com/VoltAgent/awesome-openclaw-skills) on GitHub. It lists 2,999 skills organized by category. You can also find skills on [ClawHub](https://clawhub.ai/skills).
